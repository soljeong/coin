## Affected Pages/Routes
- GET / — 메인 대시보드 (static HTML)
- GET /api/opportunities — 아비트리지 기회 목록 (수익률 순)
- GET /api/tickers — 최신 티커 데이터
- SSE /api/stream — 실시간 이벤트 스트림

## Key Interactions to Verify
- 대시보드 로드 시 opportunity 테이블 렌더링
- SSE 연결 후 3초마다 데이터 자동 갱신
- "DATA STALE" 표시 (15초 이상 데이터 미갱신 시)
- 김치 프리미엄 히트맵 정렬/필터링

## Edge Cases
- 업비트 API 다운 → 바이낸스만 표시, 크로스 거래소 기회 "STALE"
- 가격 0 또는 NaN → 해당 코인 그래프 간선 제외
- 빈 데이터 (첫 실행, DB 비어있음) → 대시보드 빈 상태 표시
- 브릿지 코인 (XRP/XLM) 모두 실패 → 환율 계산 불가 표시

## Critical Paths
- 수집기 → SQLite 저장 → 그래프 빌드 → 사이클 탐지 → API 응답 (전체 파이프라인)
- 업비트 order_books + tickers merge → bid/ask/last/volume 완성
- 벨만-포드 음수 사이클 → opportunities 테이블 저장 → 대시보드 표시
- XRP/XLM 브릿지 환율 교차 검증 → 스프레드 계산
